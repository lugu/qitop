# Improvements

## Trace mode

select a method and switch to a trace view:

        - stats: APDEX score
        - time serie: request / error rate
        - histo: call / reply size
        - histo: latency

## Debug mode

Intertwin logs with trace message base on the timestamp

        [TRACE] call ...
        [LOG D] bla bla ..
        [LOG D] bla bla ..
        [TRACE] reply ...
